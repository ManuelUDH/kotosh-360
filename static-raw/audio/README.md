# audio/

Archivos de audio para la guía auditiva multilingüe.

## Convención de nombres
`{hotspot_id}_{lang}.mp3`

Ejemplo:
- `manos_cruzadas_es.mp3`
- `manos_cruzadas_en.mp3`
- `manos_cruzadas_qu.mp3`
- `altar_central_es.mp3`
- `ambiente_kotosh.mp3`  ← audio ambiental de fondo

## Especificaciones técnicas
- Formato: MP3 128 kbps / OGG para web
- Duración máx. por clip: 90 segundos
- Locución: nativa en cada idioma
- Quechua: validado por hablantes nativos de Huánuco
