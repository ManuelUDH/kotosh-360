# ui-design/

Diseños base y guía de marca para Kotosh 360.

## Paleta de colores
| Token        | Hex       | Uso                                |
|--------------|-----------|------------------------------------|
| --accent     | #C9A84C   | Dorado inca — llamadas a la acción |
| --accent2    | #7C5CBF   | Violeta mística — highlights AR    |
| --bg         | #0F1117   | Fondo principal                    |
| --surface    | #1A1D27   | Tarjetas y paneles                 |
| --text       | #E8E8F0   | Texto principal                    |
| --muted      | #8888AA   | Texto secundario                   |

## Tipografía
- Primaria: **Inter** (UI/Dashboard)
- Secundaria: **Playfair Display** (títulos históricos en app)

## Marcadores AR físicos
Los marcadores deben imprimirse en papel resistente a la intemperie (tamaño mínimo: 15×15 cm).
Diseños en `markers/` subcarpeta.
