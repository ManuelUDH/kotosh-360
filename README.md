# Kotosh 360 🏛️
Solución de Gemelos Digitales + Realidad Aumentada para el Complejo Arqueológico de Kotosh, Huánuco.

## Módulos
- `src-backend/` — API REST (Node.js + Express + MongoDB)
- `src-dashboard/` — Panel analítico (React)
- `src-mobile-app/` — App AR (Unity C#)
- `static-raw/` — Recursos multimedia (modelos 3D, audio, diseños UI)
- `prompts/` — Prompts optimizados para generación de renders 3D
- `docs/` — Documentación UX/UI y gestión

## Inicio rápido (Backend)
```bash
cd src-backend && npm install && cp .env.example .env && npm run dev
```

## Inicio rápido (Dashboard)
```bash
cd src-dashboard && npm install && npm start
```
